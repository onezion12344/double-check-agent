# Double-Check — Intro Video

## Files

| File | Purpose |
|:-----|:--------|
| `intro.html` | 1920×1080 video page. 10 auto-advancing scenes with captions. |
| `script.md` | Narration script and scene breakdown. |
| `scene-*.mp3` | TTS voiceover segments (10 scenes), generated via MiMo V2.5 TTS (voice: Chloe). |

## How to Record

1. Open `intro.html` in a browser at fullscreen
2. Start screen recording (QuickTime Player: File → New Screen Recording)
3. Click or press → to advance manually, or let auto-advance run (7s per scene)
4. For voiceover playback: open the HTML and play the corresponding MP3 alongside
5. Stitch in editing: cut each scene to match the TTS timing

## Auto-advance Timing

Each scene auto-advances every 7 seconds. Click or arrow keys to manually control pace.
