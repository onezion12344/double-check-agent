Architecture

Two layers. One pipeline.

🔵 Hermes Plugin

Orchestrator — detects question type, injects delegate instruction. Auto-fires every turn.

🟢 Hermes Skill

Deep audit — full 4-round verification via sub-agent. On demand.

🟡 Sub-agent

Full tool access — web search, browser, terminal. Cross-verifies ≥2 sources.
